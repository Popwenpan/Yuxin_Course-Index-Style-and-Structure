A Colorful Flower

This is the first time I use CSS programming to paint. I connect HTML and CSS through a stylesheet so that the two can be associated, then I can edit HTML in the CSS interface under specified codes. For me, the HTML interface is like the initial drawing board without colouring, and CSS is the brush and paint. I can change different objects and different positions in the CSS interface to make the HTML front-end display the effect I want.

The challenge is how to edit the Chrome web interface and the CSS interface, because I adjusted the corresponding code in the Chrome interface, but forgot to copy it to the CSS interface, causing the desired effect to be reset, I am adapting to this editing method.

The flower I'm drawing now consists of three layers, six petals on the top layer, interleaved petals on the middle layer, and branches on the bottom layer. At the same time, I added a photo frame to this colourful painting and placed the photo frame on the lower layer of the petals and the upper layer of the branches, thus creating a three-dimensional feeling.

The problem I encountered now is how to change fonts with different weights because using Google fonts can only use basic fonts, I am wondering how to use CSS code to add boldly and italics fonts.
